## 它到底是个啥

简单说，PDF Squeezer 是一款 macOS 上的 PDF 压缩工具。听起来好像没什么稀奇，毕竟很多人第一反应是"这玩意 macOS 预览不是自带吗"。但真用起来你会发现，预览的"导出为 PDF→减少文件大小"那套操作，压出来的文件经常糊得没法看，而且一个一个手动导出，处理十份文件能把你点到手酸。

PDF Squeezer 干脆利落：把文件拖进窗口，或者直接拖到 Dock 图标上，压缩就开始了。三档预设——Light、Medium、Strong——一眼就能选，觉得自己是控制狂的话还能自定义压缩配置，调图片质量、分辨率、要不要删元数据这些参数都随你。

更妙的是它能在保存前并排对比压缩前后的效果。这点很关键——你压完发现文字糊了，回头还得重新压。有了对比视图，压之前就能瞄一眼，心里有数。
